# PDFテキスト抽出ツール

PDFファイルからテキストを抽出し、同じフォルダにテキストファイルとして保存するPythonツールです。

## 🌟 特徴

- **シンプルな操作**: コマンドラインでPDFファイルパスを指定するだけ
- **進捗表示**: リアルタイムでページ処理の進捗を表示
- **連続処理**: 複数のPDFファイルを連続して処理可能
- **エラーハンドリング**: 詳細なエラーメッセージと処理状況の表示
- **自動保存**: 元ファイルと同じフォルダに自動的にテキストファイルを保存

## 📋 必要な環境

- Python 3.6 以上
- インターネット接続（初回実行時のパッケージインストール用）

## 🚀 インストールと実行

### 1. リポジトリのクローン
git clone https://github.com/moracchi/pdf-text-extractor.git
cd pdf-text-extractor


### 2. プログラムの実行

python pdf_text_extractor.py


**注意**: 初回実行時に自動的にPyPDF2パッケージがインストールされます。

## 💡 使用方法

1. プログラムを実行します
2. 実行確認で `Y` を入力します
3. PDFファイルのフルパスを入力します
   - 例: `C:\Users\username\Documents\sample.pdf`
   - 例: `"C:\Users\username\Documents\sample.pdf"` （引用符付きでも可）
4. 処理完了後、以下を選択できます：
   - `1`: 別のPDFファイルを処理
   - `2`: プログラムを終了

## ⚠️ 注意事項

- スキャンされた画像PDFからはテキストを抽出できません（OCR機能は含まれていません）
- 大きなPDFファイルの処理には時間がかかる場合があります
- パスワード保護されたPDFには対応していません
- ファイルパスに日本語が含まれる場合、環境によってはエラーが発生する可能性があります

## 🐛 トラブルシューティング

### よくある問題と解決方法

**問題**: `ModuleNotFoundError: No module named 'PyPDF2'`
**解決方法**: プログラムの自動インストール機能を使用するか、手動で `pip install PyPDF2` を実行

**問題**: ファイルが見つからないエラー
**解決方法**: ファイルパスを確認し、絶対パスを使用してください

**問題**: 文字化けが発生する
**解決方法**: UTF-8エンコーディングに対応したテキストエディタを使用してください
